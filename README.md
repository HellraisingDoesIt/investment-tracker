# Investment Tracker v1.0

Personal investment portfolio tracker with real-time calculations and intelligent data management.

## Features

- **Multi-purchase tracking**: Add multiple purchases of the same ticker
- **Average cost basis**: Automatically calculates weighted average price across all purchases
- **Undo delete**: 10-second window to recover accidentally deleted positions
- **Ticker details**: Click any ticker to see grouped purchase history and totals
- **Persistent storage**: Data saved in browser localStorage

## Technologies

- HTML5
- CSS3 (Pure black theme with soft blue accents)
- Vanilla JavaScript (no frameworks)
- localStorage API

## Usage

1. Clone or download the repository
2. Open `index.html` in any modern browser
3. Add investments using the form
4. Click ticker names to see detailed purchase history
